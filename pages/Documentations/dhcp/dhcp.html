Mise en place d'un serveur DHCP sur Debian

Installez le paquet isc-dhcp-server

Mettre l'adresse du serveur en statique, ceci s'effectue dans le fichier /etc/network/interfaces

Indiquez sur laquelle de ses interface le serveur doit t'il écouter pour recevoir les requêtes dhcp. Ceci se fait dans le fichier /etc/default/isc-dhcp-server
Modifiez la ligne suivante:

INTERFACESv4="nom_interface"

Puis configurez le serveur dhcp dans le fichier /etc/dhcp/dhcpd.conf

Créez y un pool (des exemple peuvent être trouvé en commentaire dans le fichier de configuration)

N'oubliez de déclarer le réseau sur lequel se trouve le serveur, sinon le service dhcp ne fonctionnerra pas.

Une fois la configuration faites, vous pouvez démarrer le service avec la commande : "systemctl start isc-dhcp-server"
